⚙️ Firebase 초기 설정이 필요합니다

아래 단계를 따라 Firebase를 설정하면, 어느 기기에서 열어도 명단이 자동으로 동기화됩니다.

  1. 1단계 — Firebase 콘솔에서 구글 계정으로 로그인 → [프로젝트 추가] → 이름 입력 (예: classroom-seats) → 완료
  2. 2단계 — 프로젝트 홈 화면 → 웹 앱 추가(</>) → 앱 닉네임 입력 → [앱 등록] → firebaseConfig 코드 복사
  3. 3단계 — 왼쪽 메뉴 Firestore Database → [데이터베이스 만들기] → 테스트 모드로 시작 → 리전 선택(asia-northeast3) → 완료
  4. 4단계 — 이 HTML 파일을 텍스트 편집기로 열어, 아래 firebaseConfig 부분에 복사한 값을 붙여넣기
  5. 5단계 — 저장 후 브라우저에서 파일을 다시 열면 설정 완료!
